# 1. JDK 8 / 17

  • 下载 java8 地址
  • jdk 8 安装步骤【最后一项环境变量即可不配】
  • JDK 8 / 17配置不生效，配置都没错，可能是Path中关于jdk的配置优先级不够, 可置顶再试
1. 安装 jdk-8u74-windows-x64.exe
2. 安装时可以选择安装位置有两处 jdk、jer 注意： 别放同一个文件夹
2. 环境变量的配置， 在“我的电脑”中的“属性”里配置PATH、JAVA_HOME和CLASSPATH。
	path（已存在不用新建）添加变量值 			%JAVA_HOME%\bin;                                      这是你电脑里面jdk安装的bin路径
	在系统变量里新建JAVA_HOME变量，变量值为	C:\Program Files\Java\jdk1.8.0                        这是你电脑里面jdk的安装路径
	在系统变量里新建classpath变量，变量值为 	.;%JAVA_HOME%\lib\tools.jar;%JAVA_HOME%\lib\dt.jar;   这部分完整复制  
3. 在cmd窗口输入 javac 出现信息即可安装成功

# 安装前的注意点：

Maven 下载后，将 Maven 解压到一个没有中文没有空格的路径下，比如 D:\apache-maven-3.6.3 下面。解压后目录结构如下：
bin:存放了 maven 的命令，比如我们前面用到的 mvn tomcat:run
boot:存放了一些 maven 本身的引导程序，如类加载器等
conf:存放了 maven 的一些配置文件，如 setting.xml 文件
lib:存放了 maven 本身运行所需的一些 jar 包
至此，我们的 maven 软件就可以使用了，前提是你的电脑上之前已经安装并配置好了 JDK
