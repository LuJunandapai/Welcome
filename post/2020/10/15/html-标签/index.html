元素类型
	块元素 都带有"display:block/list-item"属性，p标签里无法放标签
	行内元素 display：inline 无法设置宽高，从左至右依次排列

单位
    em 相对单位，相对于父元素的字体大小
    rem 相对单位，相对于根元素的字体大小-HTML
    vw 视口宽度=100vw html的font-size的px换算成vw可等比适配
    vh 视口高度=100vh
    deg 角度单位

rem 等比例，375为设计稿设备尺寸 当前设置的css布局宽度/物理分辨率（设计稿分辨率）*基准font-size（html默认值16）
    <script>
        document.documentElement.style.fontSize=document.documentElement.clientWidth/375*16+'px'
    </script>
	
更改html默认的font-size 首选项-搜索px-修改基准font-size
样式清除 list-style：none
不透明度 opacity：0-1 0-1的范围

# 滚动

  • scrolldelay：表示滚动延迟时间，默认值为90
    • direction：表示滚动的方向，默认为从右向左

  • scrollamount: 表示移动速度,取值正整数,数值越大移动越快

# 伪类选择器

ul li:first-child{属性：属性值}
	匹配子集第一个元素
ul li:last-child{属性：属性值}
	匹配子集最后一个元素
ul li:nth-child(n){属性：属性值}
	n:索引值，从1开始
	2n、even：为偶数
	2n+1、odd：为奇数
div p:only-child{属性：属性值}
	div下有且只有一个p标签的被选中
:root{属性：属性值}
	匹配文档根元素,如：HTML
div:empty{属性：属性值}
	div里面没有任何的子元素
div:target{属性：属性值}
	目标伪类选择器，选择div的效果，添加样式
li:not(:nth-child(2n+1)){属性：属性值}
	否定伪类选择器

如何隐藏滚动条，同时仍然可以在任何元素上滚动？
    对于Firefox，我们可以将滚动条宽度设置为none： scrollbar-width: none; /* Firefox */
    对于IE，我们需要使用-ms-prefix属性定义滚动条样式： -ms-overflow-style: none; /* IE 10+ */
    对于Chrome和Safari浏览器，我们必须使用CSS滚动条选择器，然后使用display：none隐藏它：
    ::-webkit-scrollbar {
      display: none; /* Chrome Safari */
    }

# 关键帧动画

animation: 自定义名称 2s linear 1s infinite reverse forwards;
	自定义名称-动画时间-持续速度类型-延迟时间-重复次数（5/infinte-无限重复）-运动方向-运动后保持的状态
@keyframes 自定义名称{
            from/0%{ 初始属性：属性值 }
            to/100%{ 结束属性：属性值 }
          }
	可采用百分比方式，能多次设置关键点

animation-name: 自定义名称;

animation-duration: 2s; 动画时间

animation-timing-function: linear; 动画速度类型
		ease-in-out：由慢到快再到慢
		ease：平滑过渡
		linear：线性过渡
		ease-in：由慢到快
		ease-out：由快到慢
		steps(1,start/end)
		tep-start
		step-end
		1:关键帧之间的帧数，1帧为跳帧

animation-delay: 1s; 延迟时间

animation-iteration-count: 5/infinite; 动画循环次数
		infinite：无限循环

animation-direction: reverse; 循环方向
		normal：默认
		reverse：反方向
		alternate：先正常运行然后反方向、持续交替
		alternate-reverse：先反方向运行然后正常、持续交替

animation-fill-mode: forwards; 运动后的状态
		none：默认
		forwards：最后的画面
		backwards：初始画面

animation-play-state: paused; 在hover下鼠标选择下的状态
		running：运动
		paused：暂停
